# Análise de Dados Musicais de Artistas Independentes de São Paulo

Projeto desenvolvido para a Atividade Extensionista da UNINTER.

## Objetivo

Analisar dados de consumo musical de artistas independentes da cidade de São Paulo e propor estratégias acessíveis de divulgação.

## Artistas analisados

- O Terno
- Terno Rei
- Venere Vai Venus

## Tecnologias utilizadas

- Microsoft Excel
- Power BI

## Metodologia

1. Definição do público-alvo.
2. Levantamento das fontes de dados.
3. Coleta de dados de consumo musical.
4. Organização e tratamento dos dados utilizando Excel.
5. Análise dos dados.
6. Criação de visualizações e dashboards no Power BI.
7. Elaboração de recomendações.
8. Apresentação dos resultados.
